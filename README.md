## ğŸ¯ Usage

1. **Landing Page** - Click "Start" to begin building
2. **Add Nodes** - Drag node types from the right panel onto the canvas
3. **Connect Nodes** - Click and drag from one node's handle to another
4. **Edit Content** - Click any node to edit its message
5. **Save Flow** - Click "Save Flow" to validate your chatbot
6. **Export** - Use the Export menu to download as JSON or PNG
